# Customer Care Management System

## Overview
A full-stack Customer Care Management System built using the MERN stack. It allows customers to register complaints, enables administrators to assign agents, track complaint status, and manage customer records through an interactive dashboard.

## Features
- Customer Registration
- Complaint Management
- Admin Dashboard
- Agent Assignment
- Complaint Status Tracking
- Customer Management
- Responsive User Interface
- REST API Integration

## Tech Stack
### Frontend
- React.js
- Vite
- JavaScript
- HTML
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB (with local JSON fallback)

## Installation

### Clone the repository
```bash
git clone https://github.com/Monikachowdary22/Customer---Care.git
```

### Backend
```bash
cd Customer-Care/server
npm install
npm start
```

### Frontend
```bash
cd Customer-Care/client
npm install
npm run dev
```
